# remit.md SDK — DEPRECATED

> **This project has been deprecated.** remit.md has been superseded by [pay](https://pay-skill.com).

All remit.md SDKs are no longer maintained. Please migrate to the pay SDKs:

| Language | New Package | Install | Docs |
|----------|-------------|---------|------|
| Python | [`pay-sdk`](https://pypi.org/project/pay-sdk/) | `pip install pay-sdk` | [pay-skill.com/docs/sdk/python](https://pay-skill.com/docs/sdk/python) |
| TypeScript | [`@pay-skill/sdk`](https://www.npmjs.com/package/@pay-skill/sdk) | `npm install @pay-skill/sdk` | [pay-skill.com/docs/sdk/typescript](https://pay-skill.com/docs/sdk/typescript) |
| CLI | [`pay-cli`](https://crates.io/crates/pay-cli) | `cargo install pay-cli` | [pay-skill.com/docs/cli](https://pay-skill.com/docs/cli) |

## Migration

- **Website:** [pay-skill.com](https://pay-skill.com)
